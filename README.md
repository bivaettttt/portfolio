# 🌐 Portfolio (System Analyst)

Это мой персональный сайт-портфолио **системного аналитика**: компетенции, кейсы и примеры артефактов (BPMN/UML, требования, спецификации интеграций, модели данных).

## 🚀 О проекте

Сайт сделан как визитка, где удобно:
- быстро понять моё позиционирование (системный аналитик),
- посмотреть кейсы (контекст → решения → артефакты),
- перейти к материалам (PDF/MD/YAML) в репозитории.

## 🧩 Что важно для портфолио аналитика

Рекомендуемая структура артефактов:
- `cases/case-1/` — BPMN AS-IS/TO-BE, user stories, acceptance criteria
- `cases/case-2/` — sequence diagram, OpenAPI, таблица ошибок
- `cases/case-3/` — ERD, примеры SQL-запросов

## 🛠 Технологии

- HTML5 / CSS3 / JavaScript
- Тёмная/светлая тема (localStorage + prefers-color-scheme)
- Git & GitHub Pages
